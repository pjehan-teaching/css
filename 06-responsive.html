Media Queries

Les media queries permettent d'appliquer des propriétés CSS en fonction de la résolution de l'écran.

Elles peuvent aussi être utilisées en fonction du type d'écran (screen, print, tv...)

Viewport

Le viewport est la partie visible d'une page web pour l'internaute. Il peut varier en fonction de l'appareil utilisé (ordinateur, smartphone...).

La balise <meta> viewport permet d'indiquer au navigateur la façon dont il doit gérer la taille et la mise à l'échelle de la page :

<meta name="viewport" content="width=device-width, initial-scale=1.0">

width=device-width permet d'indiquer que la largeur de la page web doit correspondre à la largeur de l'écran.

initial-scale=1.0 permet de définir le niveau de zoom par défaut lorsque la page web est chargée.

Utilisation

Il est possible d'utiliser les media queries pour charger une feuille de style :

<link rel="stylesheet" media="screen and (max-width: 720px)" href="smartphone.css" />

Ou bien directement à l'intérieur d'une feuille de style :

@media screen and (max-width: 720px) {
    body {
        background-color: red;
    }
}

Les breakpoints

Les breakpoints sont des points de rupture à partir desquels une design CSS différent est appliqué.

On peut par exemple utiliser les breakpoints suivants :

// Extra small devices (portrait phones, less than 576px)
// No media query since this is the default

// Small devices (landscape phones, 576px and up)
@media (min-width: 576px) { ... }

// Medium devices (tablets, 768px and up)
@media (min-width: 768px) { ... }

// Large devices (desktops, 992px and up)
@media (min-width: 992px) { ... }

// Extra large devices (large desktops, 1200px and up)
@media (min-width: 1200px) { ... }
