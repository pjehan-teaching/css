Le positionnement

La propriété position

Par défaut, les éléments HTML suivent un flux de gauche à droite et de haut en bas.

Il est cependant possible de modifier cela en changeant la position des éléments :

Position Définition
static L'élément suit le flux normal (par défaut).
absolute L'élément est retiré du flux normal. Il est positionné relativement à son élément parent le plus proche.
fixed L'élément est retiré du flux normal. Il est positionné relativement à l'élément parent principal. Il reste toujours visible à l'écran lors du scoll
sticky L'élément suit le flux normal. Il reste toujours visible à l'écran lors du scoll.

Les éléments en dehors du flux normal sont positionnés en utilisant les propriétés top, bottom, left et right

Il faut privilégier le flux normal Exemples

Inline vs Block

Les éléments de type Inline se positionnent les uns à côté des autres tandis que les éléments de type Block on une largeur de 100% et se positionnent les uns sous les autres.

Il n'est PAS possible de définir la largeur ou la hauteur des éléments de type Inline :

<div>Block</div>
<span>Inline</span>
div, span {
    background-color: #ccc;
    height: 50px;
    width: 150px;
}
Block
Inline

Inline vs Block

Il est possible de modifier cette propriété en utilisant display :

span { display: block; }

Inline-block

Il existe également un affichage inline-block qui permet d'afficher des éléments les uns à côté des autres et de pouvoir définir leur dimension.

<ul id="demo3">
  <li>Un</li>
  <li>Deux</li>
  <li>Trois</li>
</ul>
ul#demo3 li {
    display: inline-block;
    background-color: #ccc;
}

Float

La propriété float permet de faire sortir l'élément du flux et de le rendre "flottant".

Cette propriété peut prendre plusieurs valeurs mais généralement, on utilise les valeurs left ou right.

<div id="demo4">
  <div class="square"></div>
  <p>Lorem ipsum dolor sit amet, consetetur sadipscing elitr, sed diam nonumy eirmod tempor invidunt ut labore et dolore magna aliquyam erat, sed diam voluptua.</p>
</div>
#demo4 .square {
    float: left;
}

Lorem ipsum dolor sit amet, consetetur sadipscing elitr, sed diam nonumy eirmod tempor invidunt ut labore et dolore magna aliquyam erat, sed diam voluptua.

Float

Float a depuis longtemps été utilisé pour positionner des éléments côte à côte.

float: left
float: left
float: left

Cependant, cette utilisation a été rendue obsolète avec l'arrivée de flex.

Flex

Pour répondre aux besoins de construire des pages web responsive et faciliter le positionnement des éléments, la propriété flex a été introduite dans CSS3.

Documentation CSS Tricks

Flex

Flex fonctionne avec un conteneur d'éléments qui définira comment les éléments seront positionnés.

<div id="demo-flex-1">
  <div>1</div>
  <div>2</div>
  <div>3</div>
</div>
#demo-flex-1 {
    display: flex;
}

Flex - flex-direction

La propriété flex-direction permet de choisir la direction dans laquelle les éléments seront alignés.

Les valeurs possibles sont row (défaut), column, row-reverse ou column-reverse

<div id="demo-flex-2">
  <div>1</div>
  <div>2</div>
  <div>3</div>
</div>
#demo-flex-2 {
    display: flex;
    flex-direction: column-reverse;
}

Flex - flex-wrap

La propriété flex-wrap permet de définir le comportement des éléments sortant du conteneur.

Les valeurs possibles sont nowrap (défaut), wrap ou wrap-reverse

<div id="demo-flex-3">
  <div>1</div>
  <div>2</div>
  <div>3</div>
</div>
#demo-flex-3 {
    display: flex;
    width: 150px;
    flex-direction: row;
    flex-wrap: wrap;
}

Flex - justify-content

La propriété justify-content permet de gérer l'alignement des éléments par rapport à l'axe principal (définit par flex-direction).

Les valeurs possibles sont flex-start (défaut), flex-end, center, space-between, space-around ou space-evenly

<div id="demo-flex-4">
  <div>1</div>
  <div>2</div>
  <div>3</div>
</div>
#demo-flex-4 {
    display: flex;
    justify-content: space-evenly;
}

Flex - align-items

La propriété align-items permet de gérer l'alignement des éléments par rapport à l'axe secondaire.

Les valeurs possibles sont stretch (défaut), flex-start, flex-end , center, baseline...

<div id="demo-flex-5">
  <div>1</div>
  <div>2</div>
  <div>3</div>
</div>
#demo-flex-5 {
    display: flex;
    justify-content: space-evenly;
    align-items: center;
}

Flex - align-content

La propriété align-content permet de gérer l'alignement de façon similaire à justify-content mais sur l'axe secondaire.

Les valeurs possibles sont stretch (défaut), flex-start, flex-end, center, space-between, space-around, space-evenly...

<div id="demo-flex-6">
  <div>1</div>
  <div>2</div>
  <div>3</div>
</div>
#demo-flex-6 {
    display: flex;
    width: 150px;
    flex-flow: row wrap;
    align-content: center;
}

Flex - éléments enfants

Il existe également des propriétés pour gérer le positionnement des enfants de manière individuelle :

  • order : Permet de changer l'ordre d'un élément enfant
  • flex-grow : Permet de définir si un élément enfant peut s'agrandir
  • flex-shrink : Permet de définir si un élément enfant peut rétrécir
  • flex-basis : Permet de définir la taille par défaut d'un élément enfant
  • align-self : Permet de changer l'alignement d'un élément enfant
