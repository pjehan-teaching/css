Variables (custom properties)

Depuis 2016, il est possible d'utiliser des variables en CSS :

:root {
    --main-color: #06c;
}

a {
    color: var(--main-color);
}

Variables (custom properties)

Il est également possible de déclarer une variable avec @property pour indiquer le type de valeur attendue : 

@property --main-color {
    syntax: '<color>';
    inherits: false;
    initial-value: #06c;
}

Variables (custom properties)

Il est également possible de surcharger une variable dans un sélecteur plus spécifique : 

:root {
    --main-color: lightgray;
}

body.dark {
    --main-color: darkgray;
}

Imbrication

Depuis 2023, il est possible d'utiliser l'imbrication dans le code CSS comme on peut le faire en SASS :

nav {
    ul {
        list-style-type: none;
        padding: 0;
    }
}

Il est également possible d'utiliser le symbole & pour cibler un élément parent : 

a {
    color: blue;

    &:hover {
        color: red;
    }
}

Défilement fluide

Depuis 2017, il est possible d'ajouter un défilement fluide sur une page web (particulièrement utile pour les ancres) : 

html {
    scroll-behavior: smooth;
}

Interpolation de taille

Lorsque l'on souhaite effectuer une animation lors de l'apparition d'un élément, le code suivant ne fonctionne pas : 

.box {
    height: 0;
    transition: height 0.3s;
}

.box.active {
    height: 100px;
}

Cependant, sur les navigateurs compatibles, il est possible d'utiliser l'interpolation de taille pour contourner ce problème : 

.box {
    height: 0;
    interpolate-size: allow-keywords;
    transition: height 0.3s;
}
