Le modèle de boîte

Modèle de boîte

Chaque élément du DOM est contenu dans une "boîte" CSS permettant de gérer la taille de l'élément, sa bordure ainsi que ses marges internes et externes.

Documentation MDN

Inline vs Block

Les éléments de type Inline se positionnent les uns à côté des autres tandis que les éléments de type Block on une largeur de 100% et se positionnent les uns sous les autres.

Il n'est PAS possible de définir la largeur ou la hauteur des éléments de type Inline :

<div>Block</div>
<span>Inline</span>
div, span {
    background-color: #ccc;
    height: 50px;
    width: 150px;
}
Block
Inline

Inline vs Block

Il est possible de modifier cette propriété en utilisant display :

span { display: block; }

Inline-block

Il existe également un affichage inline-block qui permet d'afficher des éléments les uns à côté des autres et de pouvoir définir leur dimension.

<ul id="demo3">
  <li>Un</li>
  <li>Deux</li>
  <li>Trois</li>
</ul>
ul#demo3 li {
    display: inline-block;
    background-color: #ccc;
}
  • Un
  • Deux
  • Trois

Modèle de boîte (inline vs block)

Les éléments de type inline n'ont pas de largeur width et de hauteur height. De plus, les propriétés margin, padding et border seront bien appliquées, mais elle ne provoqueront pas le déplacement des éléments adjacents verticalement. 

<div id="demo-model-box"  style="background-color: red;">
    <div style="display: inline; background-color: green; color: white;">
        Boîte de type inline
    </div>
    <div style="display: block; background-color: blue; color: white;">
        Boîte de type block
    </div>
</div>

#demo-model-box > * {
    margin: 10px;
    padding: 5px 10px 5px 10px;
    /* ⬆️➡️⬇️⬅️ = padding: 5px 10px; */
    border: 2px solid yellow;
}
